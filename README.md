# 📚 Python CLI Student Database

A simple, interactive Command-Line Interface (CLI) application built in Python to manage student records. This project demonstrates core programming concepts like `while` loops, conditionals, list manipulation, and dictionary data structures.

---

## ✨ Features
This program provides a menu-driven interface with the following functionalities:
* **Add Student:** Input a student's name and age to add them to the database.
* **View Students:** Display all stored student records.
* **Search Student:** Find a specific student by their name.
* **Delete Student:** Remove a student's record from the database.
* **Update Age:** Modify the age of an existing student.
* **Total Students:** Get a quick count of how many students are currently in the system.
* **Clear Database:** Delete all student records at once.

---

## 🛠️ Built With
* **Python 3.x** - Core programming language.
* **Built-in Data Structures** - Utilizes Python lists and dictionaries for in-memory data storage.
